# Agrinho-2026

🌾 Sobre o Projeto: AgroForte
O AgroForte é uma plataforma digital voltada para a agricultura de precisão e sustentabilidade. O objetivo central do projeto é unir o aumento da produtividade no campo com a redução do impacto ambiental, transformando lavouras tradicionais em ecossistemas de alta rentabilidade e de baixo carbono (carbono zero).

A plataforma foi desenvolvida utilizando tecnologias modernas de front-end para criar uma experiência de usuário imersiva, informativa e altamente interativa.

💻 O que foi desenvolvido no projeto?
O ecossistema da aplicação foi construído dividindo-se em três pilares principais de desenvolvimento técnico:

1. Interface Moderna e Responsiva (HTML5 & Tailwind CSS)
Design Fluido: Utilizando o framework Tailwind CSS, a interface foi projetada para ser totalmente responsiva, adaptando-se perfeitamente desde telas de smartphones até monitores de alta resolução.

Identidade Visual: Focada no público corporativo e do agronegócio, utilizando a tipografia Plus Jakarta Sans e uma paleta de cores baseada em tons de verde e ardósia (slate), transmitindo tecnologia e sustentabilidade.

2. Storytelling Visual com Animações Avançadas (GSAP)
Navegação por Scrollytelling: Implementação de uma seção de rolagem horizontal utilizando a biblioteca GSAP (GreenSock Animation Platform) e o plugin ScrollTrigger. Conforme o usuário rola a página para baixo, o site se move horizontalmente, guiando-o por uma jornada narrativa dividida em três fases:

Fase 01: Monitoramento por sensores IoT subterrâneos.

Fase 02: Aplicação cirúrgica de bioinsumos por drones automáticos.

Fase 03: Otimização logística para descarbonização.

3. Visualização de Dados e Simulador Interativo (Chart.js)
Gráficos em Tempo Real: Integração da biblioteca Chart.js para renderizar painéis analíticos diretamente na tela. O projeto conta com:

Gráfico de linha para monitorar a umidade crítica do solo.

Gráfico de barras comparando o desperdício de água entre pivôs tradicionais e o consumo inteligente.

Simulador de Impacto (Gráfico Radar): Um painel interativo onde o usuário clica entre "Prática Convencional" e "Solução AgroForte", e o gráfico de teia se atualiza dinamicamente, demonstrando a variação instantânea em métricas de produtividade, eficiência, margem de lucro e conservação biológica.

🛠️ Tecnologias Utilizadas
Estruturação: HTML5 Semântico.

Estilização: Tailwind CSS (Customizado via JavaScript inline para tokens de cores e fontes).

Animações: GSAP & ScrollTrigger (para controle de fluxo e efeitos de pinagem de tela).

Gráficos: Chart.js (com customização de eixos e paletas para modo escuro).

🚀 Diferencial Técnico do Código
O grande destaque do desenvolvimento foi a capacidade de integrar três bibliotecas robustas de JavaScript simultaneamente sem prejudicar a performance da página. O código separa de maneira limpa as responsabilidades de animação de tela (GSAP) da lógica de manipulação e atualização dinâmica de dados de simulação (Chart.js), resultando em uma aplicação rica em micro-interações e de fácil manutenção.
